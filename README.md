# Luxshan Thuraisingam | Portfolio

A modern, responsive portfolio website built to showcase my software engineering projects, skills, and experience.

## Tech Stack
- **Frontend:** React, Vite
- **Styling:** CSS (Custom Variables, Flexbox/Grid)
- **Deployment:** Vercel / GitHub Pages (Ready)

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```
